
## Ionic







### Requirements

**Framework Setup:**
  
ionic: https://ionicframework.com/




### Run Locally

Clone the project

```bash
  git clone https://github.com/yosuva-rajendran/ionic.git
```

Go to the project directory

```bash
  cd ionic-base
```

Install dependencies

```bash
  npm install 
```

Start the server

```bash
  ionic serve
```
